# 📊 Natural Gas Price Forecasting

## 📌 Project Overview
This project analyzes historical natural gas prices and builds a forecasting model to estimate future prices.

## 🎯 Objectives
- Analyze price trends
- Identify seasonality patterns
- Predict prices for future dates

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib

## 📈 Key Insights
- Prices show an upward trend over time
- Seasonal variation observed:
  - Higher in winter
  - Lower in summer

## 🔮 Model Approach
- Linear regression for trend
- Monthly averaging for seasonality
